## HTML5的新特性
### HTML新增的语义标签
`<header>`、`nav`、`article`、`section`、`aside`、`footer`标签有语义，有助于浏览器的搜索引擎搜索，以上标签均为块级元素。
### 媒体标签
#### 视频标签
尽量使用mp4格式
```html
<video src="文件地址" controls="controls"></video>
<!-- 或者 -->
<video controls="controls" width="300"> 
  <source src="move.ogg" type="video/ogg" > 
  <source src="move.mp4" type="video/mp4" > 
  您的浏览器暂不支持 <video> 标签播放视频
</video>
```
**视频标签属性**：
- autoplay(自动播放)
- controls
- width
- height
- loop(循环播放)
- src(播放源)
- muted(静音播放)
- poster(等待加载时的图片)

> 在谷歌浏览器，默认静止了自动播放，如果想要自动播放效果，需要设置muted属性

#### 音频标签
```html
<audio src="文件地址" controls="controls"></audio>
<!-- 或者 -->
< audio controls="controls" >
  <source src="happy.mp3" type="audio/mpeg" >
  <source src="happy.ogg" type="audio/ogg" > 
  您的浏览器暂不支持 <audio> 标签。
</audio>
```
**音频标签属性**：
- autoplay(自动播放)
- controls
- loop(循环播放)
- src(播放源)

> 在谷歌浏览器，默认静止了自动播放，如果想要自动播放效果，后面可以通过JavaScript来制作。

### 表单
#### 新增的表单类型
重点记住：
- type="number"
- type="tel"
- type="search"

#### 新增的表单属性
- required(表示其内容不能为空)
- placeholder(表单的提示信息)
- autofocus(自动聚焦属性)
- autocomplete(自动填充、默认打开)
- multiple(可以多选文件提交)

可更改placeholder里面的字体颜色：
```css
input::placeholder {
  color: blue;
}
```
## CSS3的新特性
### 新增选择器
#### 属性选择器
- E[alt] 选择具有alt属性的元素
- E[alt="val"] 选择具有alt属性等于val的元素
- E[alt^="val"] 选择具有alt属性且值以val开头的的元素
- E[alt$="val"] 选择具有alt属性且值以val结尾的的元素
- E[alt*="val"] 选择具有alt属性且值中含有val的的元素

> 类选择器、属性选择器、伪类选择器，权重为10

#### 结构伪类选择器
#### 伪元素选择器
- ::before
- ::after

注意：
- 1、before、after创建一个行内元素
- 2、新创建的元素在DOM树中找不到（伪元素）
- 3、必须具有content属性
- 4、伪元素和标签一样，权重为1

```css
/* 权重为1+1 */
p::befor {
  content: '';
  position: absolute;
  top: 10px;
  left: 20px;
}
```
### 盒子模型
- `box-sizing: content-box;` 盒子大小为 width+padding+border
- `box-sizing: border-box;` 盒子大小为 width

```

```
### 过渡
过渡（transition）动画：从一个状态到渐渐过渡到另一个状态，经常和`:hover` 一起搭配，可见[demo](/demo/过渡.html)
  
transition: 要过渡的属性 花费时间 运动曲线 何时开始;   
- 1、属性: 想要变化的css属性，宽度高度、背景颜色、内外边距都可以。如果想要所有的属性都
变化过渡，写一个all就可以。
- 2、运动曲线：默认是ease（逐渐慢下来）、linear（匀速）、ease-in（加速）、ease-out（减速）、ease-in-out（先加速后减速）
- 3、何时开始：默认是0s

> 过度的秘诀：谁做过渡给谁加

### 2D转换
转换（transform）是CSS3中具有颠覆性的特征之一，可实现元素的位移、旋转、缩放等效果。
- translate(移动)
- rotate(旋转)
- scale(缩放)

#### translate
2D移动是2D转换里面的一种功能，可以改变元素在页面中的位置，类似**定位**。
```css
  transform: translate(x,y); 
  /* 或者分开写  */
  transform: translateX(n); 
  transform: translateY(n);
```

特点：
- 不会影响到其他元素的位置
- translate中的百分比单位是相对于自身元素的translate:(50%, 50%);
- 对行内标签无效

#### rotate
2D旋转指的是让元素在2维平面内顺时针旋转或者逆时针旋转。可见[demo](/demo/rotate.html)
```css
transform:rotate(度数);
```

特点：
- rotate里面跟度数，单位是deg 比如`rotate(45deg)`;
- 角度为正，顺时针；角度为负，逆时针；
- 默认旋转的中心点是元素的中心点

`transform-origin`可以设置元素转换的中心点。可见[demo](/demo/rotate_origin.html)
```css
transform-origin: x y;
```
特点：
- x 和 y用空格隔开
- 默认中心点是元素的中心点(50%, 50%)
- 还可以给x y设置像素 或者方位名词(top bottom left right center)

#### scale
缩放，可以控制放大和缩小
```css
transform: scale(x, y);
```
特点：
- x和y用逗号隔开
- `transform: scale(2);` 只写一个参数，第二个参数和第一个参数一样，相当于`scale(2, 2);`
- scale 缩放的最大优势： 可以设置转换中心点，默认以中心点缩放，不影响其他盒子

分页按钮缩放案例可见[demo](/demo/分页按钮缩放.html)
#### 2D转换写法
```css
transform: translate() rotate() scale(); 
/* 顺序会影响结果 */
/* 当有位移和其他属性时，记得将位移放在最前面 */
```
### 动画
动画(animation)，可以通过多个节点控制一个或一组动画，常用来实现复杂的动画效果。  
动画的基本使用：  
- 1、先定义动画
- 2、再使用动画

#### 定义动画
用keyframes定义动画(类似定义类选择器)
```css
@keyframes move {
  /* 0%是动画的开始，100%是动画的完成 */
  0% {
    width: 100px;
  }
  100% {
    width: 200px;
  }
  /* 也可以使用from ... to 来表示 */
}
```

#### 元素使用动画
```css
div {
  width: 200px;
  height: 200px;
  background-color: aqua;
  /* 调用动画 */
  animation-name: 动画名字;
  /* 持续时间 */
  animation-duration: 持续时间;
}
```
#### 动画属性
- `@keyframes` 规定动画
- `animation` 所有动画属性的简写属性， 除了animation-play-state属性
- `animation-name` 规定@keyframes动画的名称（必须的）
- `animation-duration` 规定动画完成一个周期所花费的时间，默认是0（必须的）
- `animation-timing-function` 规定动画的速度曲线，默认是"ease"
  - `linear` 匀速
  - `ease` 默认、低速开始，然后加速、最后变慢
  - `ease-in` 低速开始
  - `ease-out` 低速结束
  - `ease-in-out` 低速开始和结束
  - `steps()` 指定时间函数中的间隔数量(步长)
- `animation-delay` 规定动画何时开始，默认是0
- `animation-iteration(重复的)-count` 规定动画被播放的次数，默认是1，还有infinite(无限的)
- `animation-direction` 规定动画是否在下一周期逆向播放，默认是normal、alternate是逆播放 
- `animation-play-state` 规定动画是否正在运行或暂停，默认是running，还有paused
- `animation-fill-mode` 规定动画结束后状态，保持forwards回到起始状态backwards

#### 动画简写
```css
animation: 动画名称 持续时间 运动曲线 何时开始 播放次数 是否反方向 动画起始或者结束状态
```

### 3D转换
#### translate3D
- translateX(100px)
- translateY(100px)
- translateZ(100px)
- translate3d(100px, 100px, 100px)

[demo](/demo/translate3d.html)
#### 透视(perspective)
透视我们也称为视距，视距越大，图像越小，透视需要写在观察盒子的**父盒子**上面或者**祖父盒子**。
#### 旋转(rotate3d)
- rotateX(45deg)
- rotateY(45deg)
- rotateZ(45deg)
- rotate3d(1, 0, 0, 45deg)(参数可更改)了解即可

#### transform-style
- 控制子元素是否开启三维立体环境
- `transform-style: flat;` (flat水平的)子元素不开启3d立体空间，默认的
- `transform-style: preserve-3d;` (preserve保持)子元素开启立体空间
- 代码**写给父级**，但是影响的是子盒子
