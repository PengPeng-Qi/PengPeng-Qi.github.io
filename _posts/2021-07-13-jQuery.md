## jQuery入门
### jQuery概述
`j`：JavaScript、`Query`：查询；对原生js做了封装，可以快速查询使用里面的功能。
### jQuery基本使用
jQuery的顶级对象`$`、相当于原生JavaScript中的`window`，`$`是jQuery的别称。  
### DOM对象与jQuery对象
用原生JS获取来的对象是DOM对象。  
jQuery方法获取的元素就是jQuery对象。  
jQuery对象本质是：利用$对DOM对象包装后产生的对象（伪数组形式存储）  
```js
  // 1. DOM 对象：  用原生js获取过来的对象就是DOM对象
  var myDiv = document.querySelector('div'); // myDiv 是DOM对象
  var mySpan = document.querySelector('span'); // mySpan 是DOM对象
  console.dir(myDiv);
  // 2. jQuery对象： 用jquery方式获取过来的对象是jQuery对象。 本质：通过$把DOM元素进行了包装
  $('div'); // $('div')是一个jQuery 对象
  $('span'); // $('span')是一个jQuery 对象
  console.dir($('div'));
  // 3. jQuery 对象只能使用 jQuery 方法，DOM 对象则使用原生的 JavaScirpt 属性和方法
  // myDiv.style.display = 'none';
  // myDiv.hide(); myDiv是一个dom对象不能使用 jquery里面的hide方法
  // $('div').style.display = 'none'; 这个$('div')是一个jQuery对象不能使用原生js 的属性和方法
```
#### jQuery对象与DOM对象的转换
原生`js`比`jQuery`对象大。**DOM对象转换为jQuery对象**的方法：`$('div')`。  
**jQuery对象转换为DOM对象**：
- `$('div')[index]`、index是索引号
- `$('div').get(index)`  

```html
<video src="mov.mp4" muted></video>
<script>
    // 1. DOM对象转换为jQuery对象
    // (1) 我们直接获取视频，得到就是jQuery对象
    // $('video');
    // (2) 我们已经使用原生js 获取过来 DOM对象
    var myvideo = document.querySelector('video');
    // myvideo.play();  DOM对象
    // $(myvideo).hide();  DOM里面没有hide这个方法，所以转换为了jQuery对象，注意()没有引号

    // 2.  jQuery对象转换为DOM对象
    // $('video')[0].play()
    $('video').get(0).play()
</script>
```
## jQuery常用API
### jQuery选择器
`$('选择器')`
#### 隐式迭代
遍历内部DOM元素（伪数组形式存储）的过程叫做**隐式迭代**。
```html
<div>1</div>
<div>1</div>
<div>1</div>
<div>1</div>
<ul>
  <li>2</li>
  <li>2</li>
  <li>2</li>
</ul>
<script>
  // 1. 获取四个div元素 
  console.log($("div"));

  // 2. 给四个div设置背景颜色为粉色 jquery对象不能使用style
  $("div").css("background", "pink");
  // 3. 隐式迭代就是把匹配的所有元素内部进行遍历循环，给每一个元素添加css这个方法
  $("ul li").css("color", "red");
</script>
```
#### 筛选选择器
- `:first`获取第一个li元素
- `:last`获取最后一个li元素
- `eq(index)`获取到的li元素中，选择索引号为2的元素，索引号index从0开始`$('ul li:eq(2)')`不推荐这个写法
- `:odd`获取到的li元素中，选择索引号为奇数的元素
- `:even`获取到的li元素中，选择索引号为偶数的元素
- `hasClass('类名')`：检测当前的元素是否含有特定的类，如果有则返回true

```html
<script>
  $(function() {
    $("ul li:first").css("color", "red");
    $("ul li:eq(2)").css("color", "blue");
    $("ol li:odd").css("color", "skyblue");
    $("ol li:even").css("color", "pink");
  })
</script>
```
#### 筛选方法
- `parent()`：查找父级
- `children('选择器')`：子代选择器
- `find('选择器')`：后代选择器
- `siblings('选择器')`：兄弟节点，不包括自己
- `nextALL([exr])`：查找当前元素之后所有的同辈元素
- `prevALL([exr])`：查找当前元素之前所有的同辈元素
- `eq(index)`：第n个元素，是选择方法`$('ul li').eq(2)`，更推荐这个方法


```js
// 注意一下都是方法,带括号
$(function() {
    // 1. 父 parent() 返回的是最近一级的父级元素
    console.log($(".son").parent());
    // 2. 子
    // (1)children(), 类似子代选择器, ul>li
    $(".nav").children("p").css("color", "red");
    // (2)可以选里面所有的孩子 find() 类似于后代选择器
    $(".nav").find("p").css("color", "red");
});
```
```html
<ul class="nav">
  <li>
    <a href="#">微博</a>
    <ul>
      <li><a href="">私信</a></li>
      <li><a href="">评论</a></li>
      <li><a href="">@我</a></li>
    </ul>
  </li>
  <li>
    <a href="#">微博</a>
    <ul>
      <li><a href="">私信</a></li>
      <li><a href="">评论</a></li>
      <li><a href="">@我</a></li>

<script>
  $(function() {

```
排他思想：
```js
$(function() {
  // 1. 隐式迭代 给所有的按钮都绑定了点击事件
  $("button").click(function() {
    // 2. 当前的元素变化背景颜色
    $(this).css("background", "pink");
    // 3. 其余的兄弟去掉背景颜色 隐式迭代
    $(this).siblings("button").css("background", "");
  });
})
```

### 链式编程
链式编程是为了节省代码量
```js
$(this).css('color', 'red');
$(this).siblings.css('color', '');

// 等同于
$(this).css('color', 'red').sibling().css('color', '');
```
### jQuery样式操作
#### 操作css方法
可以使用css方法来修改简单元素样式，也可以操作类，修改多个样式。  
```js
//  1.参数只写属性名，则返回属性值 
$(this).css('color')；

//  2.参数是属性名，属性值，逗号分割，属性必须加引号，值如果是数值可以不加单位和引号
$(this).css('color', 'red');

// 3.参数可以是对象形式，方便设置多组样式，属性名和属性值用冒号隔开，属性可以不用加引号
$("div").css({
    width: 400,
    height: 400,
    backgroundColor: "red"
    // 如果是复合属性则必须采取驼峰命名法，如果值不是数字，则需要加引号
})
```
#### 设置类样式方法
添加类`$('选择器').addClass('类名')`  
删除类`$('选择器').removeClass('类名')`  
切换类`$('选择器').toggleClass('类名')`  
```js
$("div").click(function() {
    $(this).toggleClass("current");
});
```
### jQuery效果
- 显示隐藏：
  - `show([speed, [easing], [fn]])`
    - `speed`：`slow`、`normal`、`fast`、或表示动画时长的毫秒数
    - `easing`：默认`swing`、可选`linear`
    - `fn`：回掉函数
  - `hide()`
  - `toggle()`
- 滑动：
  - `slideDown()`
    - `speed`：`slow`、`normal`、`fast`、或表示动画时长的毫秒数
    - `easing`：默认`swing`、可选`linear`
    - `fn`：回掉函数
  - `slideUp()`
  - `slideTpggle()`
- 淡入淡出：`fadeln()`、`fadeOut()`、`fadeToggle()`、`fadeTo()`
  - `fadeTo()`用法，`fadeTo([speed, opacity, [easing], [fn]])`
    - `opacity`透明度必须写，取之0～1
    - `speed`必须写
- 自定义动画：`animate(params, [speed], [easing], [fn])`
  - `params`：想要更改的样式，必须写，属性名可以不带引号，如果是符合属性则需要采用驼峰命名法，其余参数可以省略

**事件切换**：`hover([over,]out)`
- over鼠标经过时的函数
- out鼠标离开时的函数

```js
$(".nav>li").hover(function() {
    $(this).children("ul").slideToggle();
});
```
**动画排队**：动画或效果一旦触发就会执行，如果多次触发，就造成多个动画或者效果排队执行。`stop()`方法用于停止动画或效果。    
```js
$(".nav>li").hover(function() {
    $(this).children("ul").stop().slideToggle();
});
```

### jQuery属性操作
#### 获取属性值
获取元素自带属性值`element.prop('属性名')`  
设置元素自定义属性值`element.attr('属性名')`
```js
$('div').prop('class');

$('div').attr('class');
```

#### 设置属性值
```js
// 设置自带属性值
$('a').prop('href', 'www.baidu.com');

// 设置自定义属性
$('index').attr('href', 'www.baidu.com');
```
